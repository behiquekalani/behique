# Shrine Theme License Analysis
# Researched: 2026-03-21

## What We Know
- 1 license = 1 store at a time
- Can transfer between stores (not simultaneous)
- Need additional licenses for multiple stores
- Source: https://shrine.io/pages/faq

## What's NOT Stated
- No explicit permission to resell modified themes
- No explicit permission to create derivative works for sale
- No mention of developer/reseller license tier

## Risk Assessment
SELLING MODIFIED SHRINE THEMES ON GUMROAD = HIGH RISK without explicit permission.
Standard theme licenses in the Shopify ecosystem do NOT allow resale of derivatives.

## Safe Alternatives

### Option A: Sell as a SERVICE (not a theme)
- "Shopify Store Setup Service" - $49/$99/$149
- You use YOUR Shrine license to build their store
- You're selling your time and expertise, not the theme files
- LEGAL: Yes, this is like a web designer using licensed software

### Option B: Build on Dawn (MIT License, fully legal to resell)
- Shopify's official open-source theme
- MIT License = modify, distribute, sell, no restrictions
- Rebuild the Behike designs on Dawn's codebase
- Takes more work but zero legal risk

### Option C: Contact Shrine for developer license
- Email: support@shrine.io or their Discord
- Ask: "Do you offer a developer/reseller license for selling customized versions?"
- Some theme developers offer this for $500-$2000

## Recommendation
Use Option A (service) for immediate revenue.
Build on Dawn (Option B) for Gumroad theme products long-term.
The 3 Shrine-based themes we built can be used as DESIGN REFERENCES
but should not be sold as downloadable theme files.
