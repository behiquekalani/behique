## HOW TO USE THIS DOCUMENT

This timeline maps git commits to the creation of specific products and brand assets. Each commit has a cryptographic SHA hash and timestamp that cannot be altered retroactively without breaking the hash chain. This makes git history strong evidence of creation dates.

To verify any entry, run:
```bash
cd ~/behique
git log --all --oneline --date=iso --format="%H %ad %s" | grep "[search term]"
```

## HOW TO EXPORT FULL GIT LOG AS EVIDENCE

Run this command to create a complete, dated record of all commits:

```bash
cd ~/behique
git log --all --date=iso --format="COMMIT %H%nDATE: %ad%nAUTHOR: %an <%ae>%nSUBJECT: %s%n" > ~/behique/legal/evidence/full-git-log-$(date +%Y-%m-%d).txt
```

This creates a timestamped export of the entire commit history that can be submitted as evidence of creation dates. The SHA hashes are cryptographic proof that the history has not been altered.
