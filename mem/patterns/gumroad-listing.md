# Gumroad Listing Template

## GUMROAD_INFO.txt Format
```
GUMROAD LISTING INFO
====================

Name: [Product Name]
Price: [$X.99]
URL slug: [short-slug]
Call to action: Buy this
Category: Self Improvement > Productivity
Tags: [5-6 relevant tags, comma separated]
Receipt button: Download [Product Type]

DESCRIPTION:
[Hormozi-style copy. Pain → system → proof → what's inside.
2-4 short paragraphs. No fluff. See copywriting-voice pattern.]

SUMMARY (You'll get...):
1 PDF [type] ([X] modules/lessons/pages)
[Key deliverable 1]
[Key deliverable 2]
[Key deliverable 3]

RECEIPT MESSAGE:
Welcome to Behike. [Specific action]. behike.gumroad.com
```

## Pricing Tiers
- Free lead magnet: $0 (Starter Blueprint)
- Tutorial guides: $4.99 - $9.99
- Blueprints: $19.99
- Voice Bible: $16.99
- Blueprint Bundle: $49.00
- Behike OS: $97.00

## Discount Code
- BEHIKE10 = 10% off everything

## URL Slug Rules
- Short, memorable, lowercase
- Match the product concept: `youtube`, `freelancer`, `first-dollar`
- Spanish variants: same slug + `-es` suffix
- No version numbers in slugs

## Tags Strategy
- Always include: the niche keyword, "blueprint" or "guide", format type
- Include adjacent keywords: "AI", "automation", "solopreneur"
- 5-6 tags max

## Category
- Almost everything: Self Improvement > Productivity
- Tech guides: could also use Technology
