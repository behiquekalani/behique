# GUI Skins — Template Archive

Every UI Ceiba builds gets saved here as a reusable skin. These can be swapped, remixed, or used as starting points for new interfaces.

## Available Skins

| Skin | File | Aesthetic | Notes |
|------|------|-----------|-------|
| Pip-Boy Quest Log | `pipboy-quest-log.html` | Fallout terminal, green-on-black, Press Start 2P | Original quest dashboard v1.1 |
| Ceiba Face Wireframe | `ceiba-face-wireframe.html` | Wireframe grid head, green hologram, scanlines | 4 expressions, CRT overlay |

## How to Use
Each skin is a self-contained HTML file. The active HQ is always `quest-dashboard.html` in the repo root. To swap skins, copy the desired template over or build a theme-switcher into the HQ.
