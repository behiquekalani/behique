Behike Gumroad Product Covers

Each cover below is exactly 1280 x 720px, the recommended Gumroad thumbnail size.

How to Screenshot

  1. Open this file in Chrome or Safari
  2. Right-click a cover and choose Inspect
  3. In DevTools, click the .cover div in the Elements panel
  4. Right-click the node and select Capture node screenshot
  5. The exported PNG will be exactly 1280x720

Alternative: macOS Screenshot

  1. Zoom browser to 100% (Cmd + 0)
  2. Use Cmd + Shift + 4, then press Space to capture the window
  3. Or use a tool like CleanShot X to capture a scrolling area at exact dimensions

Programmatic Export

You can also use Puppeteer or Playwright to screenshot each .cover div automatically:

page.locator('.cover').nth(0).screenshot({ path: 'cover-1.png' })
