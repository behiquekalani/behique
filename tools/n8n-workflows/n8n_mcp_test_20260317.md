## Notes on the eBay MCP Server Suite

The complete eBay MCP suite (templates 5533-5542) is built by the same verified creator. Each template:
- Uses an **MCP Trigger node** as the server endpoint
- Wraps eBay REST API calls in **HTTP Request nodes**
- Uses `$fromAI()` expressions so AI agents auto-populate parameters
- Returns native eBay API responses

This means once imported into Cobo's n8n, any AI agent (Claude/Ceiba, GPT, etc.) could connect via MCP and perform eBay operations through natural language. This is the most powerful finding from this exploration.
