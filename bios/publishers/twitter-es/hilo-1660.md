# Hilo 1660

## Tweet 1

## Tweet 2
No es cuánto quieres ganar. Es cuánto necesitas para vivir exactamente como quieres. Ese número tiene gastos fijos, variables, ahorro y margen. Es concreto.

## Tweet 3
Con ese número sabes exactamente cuántos clientes necesitas, cuántos productos tienes que vender, si el modelo actual puede llevarte ahí.
