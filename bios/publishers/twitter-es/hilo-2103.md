# Hilo 2103: Lista de Email

**Tweet 1:** Tu lista de email vale más que todos tus seguidores en redes juntos. No es hipérbole. Es matemática de acceso y conversión.

**Tweet 2:** Instagram puede cambiar el algoritmo. TikTok puede desaparecer. Twitter puede colapsar. Tu lista no le pertenece a ninguna plataforma. Es tuya.

**Tweet 3:** Mil suscriptores comprometidos generan más ingresos que cien mil seguidores pasivos. La lista es el activo. Las redes son el canal de descubrimiento.

**Tweet 4:** Cada publicación en redes debería tener un camino hacia la lista. Cada pieza de valor debería tener una puerta de entrada. Las redes para el descubrimiento. El email para la relación.

**Tweet 5:** La lista que crece sola es la que recibe valor real con frecuencia sostenible. Define qué valor entregarás. Define la frecuencia. Cumple eso sin excepción.
