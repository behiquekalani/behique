# Hilo 2198: Precios y valor percibido

**Tweet 1:** El precio bajo no atrae mejores clientes. Atrae más problemas. Los clientes de precio bajo son los más exigentes y los que menos valoran el trabajo.

**Tweet 2:** El precio es una señal de valor. Cuando subes el precio, filtras. Dices: esto es para alguien que toma el proceso en serio. Ese filtro cambia quién llega.

**Tweet 3:** El precio correcto no se calcula desde tus costos. Se calcula desde el valor del cliente. Cuánto vale para ellos resolver este problema. Ese número es muy diferente a tus costos más margen.

**Tweet 4:** Un solopreneur vendiendo a $200 necesita 50 ventas para $10,000. Vendiendo a $1,000 necesita 10. Vendiendo a $5,000 necesita 2. El volumen no es el único camino.

**Tweet 5:** Sube el precio. Mejora la oferta. Trabaja con menos personas mejor alineadas. El negocio small-and-profitable es un negocio real.
