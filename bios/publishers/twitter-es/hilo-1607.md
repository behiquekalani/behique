# Hilo 1607

## Tweet 1
Sistemas vs hábitos: la diferencia que cambia si construyes algo duradero o si dependes de cómo amaneciste.

## Tweet 2
El hábito requiere motivación para activarse. El sistema tiene un proceso que sigue aunque la motivación no esté.

## Tweet 3
El hábito dice: publico cuando tengo ideas. El sistema dice: cada martes a las 9am abro el documento de ideas y elijo una.

## Tweet 4
El hábito se rompe cuando la vida se complica. El sistema sobrevive porque está diseñado para los días difíciles, no los fáciles.

## Tweet 5
No construyas hábitos. Construye sistemas con hábitos dentro. El sistema es el contenedor. Los hábitos son las piezas.
