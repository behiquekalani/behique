# Hilo 2115: Sistemas vs motivación

**Tweet 1:** Dependes de la motivación para trabajar? Ese es el problema. La motivación es un recurso renovable con ciclos impredecibles. El sistema es infraestructura.

**Tweet 2:** La diferencia es simple. La motivación llega cuando quiere. El sistema funciona porque lo diseñaste para funcionar aunque no quieras. Los dos producen trabajo. Solo uno produce trabajo consistente.

**Tweet 3:** Diseñar un sistema no significa ser robot. Significa decidir de antemano cuándo, dónde, cuánto, y cómo trabajas en cada cosa. Esas decisiones tomadas de antemano no gastan energía el día que las ejecutas.

**Tweet 4:** El día malo con sistema produce un resultado mínimo viable. El día malo sin sistema produce nada. Multiplicado por trescientas sesenta y cinco días, la diferencia es la distancia entre donde estás y donde quieres estar.

**Tweet 5:** Diseña el sistema esta semana. No esperes motivación para hacerlo. Eso sería el problema dentro del problema.
