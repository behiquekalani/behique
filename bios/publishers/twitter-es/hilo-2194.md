# Hilo 2194: Sistemas vs motivación

**Tweet 1:** La motivación es un estado de ánimo. Los sistemas son permanentes. Si dependes de motivación para trabajar, estás construyendo sobre arena.

**Tweet 2:** Un sistema no necesita que estés inspirado. Necesita que sigas el proceso. Lunes: ideas. Martes: desarrollo. Miércoles: publicación. El sistema decide, tú ejecutas.

**Tweet 3:** La motivación llega como consecuencia del progreso, no como condición para empezar. Actúa primero. La motivación llega cuando ves movimiento.

**Tweet 4:** Los sistemas tienen otro beneficio: eliminan las decisiones repetitivas. Cuando no tienes que decidir cuándo escribir, usas esa energía en qué escribir. El foco mejora.

**Tweet 5:** Diseña el sistema más simple que funcione. Toma el primer paso hoy, aunque no tengas ganas. El sistema se consolida con uso. La motivación viene después.
