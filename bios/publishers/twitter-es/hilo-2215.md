# Hilo 2215: Sistemas vs Motivación

**Tweet 1:** La motivación te saca de la cama. El sistema te mantiene trabajando cuando la motivación no llega.

**Tweet 2:** Depender de cómo te sientes para producir es inestable. Los creadores exitosos no esperan inspiración. Tienen un sistema que produce sin ella.

**Tweet 3:** Un sistema no tiene que ser complejo. Puede ser un template, un horario, un checklist. Lo que importa es que funcione cuando no tienes ganas.

**Tweet 4:** La motivación es combustible. Se agota. El sistema es el motor. Sigue funcionando aunque el combustible sea bajo.

**Tweet 5:** Diseña tu sistema para tu peor día. Si funciona ese día, funciona siempre.
