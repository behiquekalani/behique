# Hilo 1986

## Tweet 1
Los productos digitales no son un ingreso extra. Son un modelo de negocio. Si los tratas como extras, seguirán siendo extras.

## Tweet 2
El nivel de seriedad con el que construyes determina el nivel de resultados que obtienes. Eso no tiene excepción.

## Tweet 3
Trata tu primer producto como si fuera el primero de cien. Construye el sistema de entrega. La página de ventas. El proceso post-compra.

## Tweet 4
La diferencia entre el creador que gana y el que no no es el producto. Es el sistema alrededor del producto.

## Tweet 5
Construye el negocio, no solo el producto. El producto es el 20%. El sistema es el 80%.
