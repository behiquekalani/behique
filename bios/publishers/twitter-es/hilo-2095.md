# Hilo 2095: Sistemas vs Motivación

**Tweet 1:** La motivación es un estado emocional. Fluctúa. Los sistemas no. La diferencia entre el creador consistente y el inconsistente no es motivación. Es infraestructura.

**Tweet 2:** Un sistema bien diseñado funciona el día que dormiste mal, el día que el resultado fue decepcionante, el día que simplemente no tienes ganas.

**Tweet 3:** La trampa es esperar sentirte motivado para actuar. La motivación viene después de empezar. El sistema te da el inicio sin depender de ese estado.

**Tweet 4:** Diseña el proceso para que requiera la menor cantidad de fuerza de voluntad posible. Reduce fricción. Crea condiciones. Automatiza lo que puedas.

**Tweet 5:** El sistema hace el trabajo. Tú solo tienes que aparecer. Y aparecer es mucho más fácil cuando el sistema está esperándote.
