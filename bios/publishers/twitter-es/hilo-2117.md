# Hilo 2117: Aprender ejecutando, no acumulando

**Tweet 1:** Hay dos maneras de aprender. Acumular conocimiento antes de actuar. O actuar y aprender en el proceso. Solo una funciona para construir negocios reales.

**Tweet 2:** El conocimiento acumulado sin contexto de aplicación se olvida rápido y se usa mal. El conocimiento adquirido mientras resuelves un problema real se integra profundamente porque tienes el contexto.

**Tweet 3:** El ciclo que funciona: actúa con lo que sabes. Encuentra el obstáculo. Aprende específicamente lo que necesitas para superar ese obstáculo. Actúa de nuevo. El aprendizaje tiene dirección y propósito.

**Tweet 4:** El ciclo que no funciona: estudia hasta sentirte listo. Busca más recursos cuando sientes dudas. Postpone la acción porque falta preparación. Nunca hay suficiente preparación cuando el estándar es la certeza.

**Tweet 5:** Lanza con el 70% de preparación que tienes hoy. El 30% restante lo aprendes en el camino. Esto no es imprudencia. Es la manera en que funciona el aprendizaje real.
