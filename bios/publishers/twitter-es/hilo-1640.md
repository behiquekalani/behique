# Hilo 1640

## Tweet 1
Consistencia gana sobre intensidad.

Siempre. Sin excepciones. A largo plazo.

## Tweet 2
Un mes de trabajo intenso seguido de tres meses de silencio no construye audiencia.

No construye confianza. No construye hábito en el lector.

## Tweet 3
El creador que publica con regularidad durante dos años llega a lugares que el creador de rachas nunca alcanza.

No porque sea más brillante. Porque estuvo ahí más tiempo.

## Tweet 4
La consistencia no requiere que cada pieza sea brillante.

Requiere que cada pieza exista. El brillante aparece dentro del volumen.

## Tweet 5
Diseña el sistema para el peor día, no para el mejor.

El sistema que dura es el que puedes mantener cuando estás cansado, distraído, sin inspiración.

Ese es el sistema real.
