# Hilo 1434

## Tweet 1
El nicho no te limita. Te posiciona.

Cuando eres el mejor recurso para un problema especifico, te vuelves indispensable para las personas con ese problema.

## Tweet 2
El error al elegir nicho:

Elegir por passion unicamente.
"Me gusta el fitness" no es nicho.
"Ayudo a hombres de 35 a 45 anos que quieren volver a entrenar despues de una lesion" si es nicho.

## Tweet 3
El nicho correcto tiene tres elementos:

Problema especifico que duele.
Audiencia que tiene ese problema y puede pagar.
Tu que tienes la experiencia o conocimiento para resolverlo.

Los tres juntos. No dos de tres.

## Tweet 4
Una vez tienes el nicho, todo se simplifica:

Sabes para quien escribes.
Sabes que problema resuelves.
Sabes que producto crear.

La claridad del nicho es la claridad del negocio.

## Tweet 5
El nicho puede evolucionar con el tiempo.

Empiezas enfocado. Con el tiempo amplias desde esa base de autoridad.
Pero empezar amplio para ver que pega es la estrategia mas lenta y mas agotadora.

Empieza estrecho. Profundiza. Expande despues.
