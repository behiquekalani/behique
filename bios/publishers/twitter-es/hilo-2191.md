# Hilo 2191: La plataforma es el camino, no el destino

**Tweet 1:** El error más común en redes sociales: construir para la plataforma en vez de para la persona.

**Tweet 2:** Si construyes para el algoritmo, cuando el algoritmo cambia pierdes todo. Si construyes para la persona, la llevas contigo a donde sea que vayas.

**Tweet 3:** Instagram, TikTok, LinkedIn son herramientas de descubrimiento. No son tu negocio. Son el camino hacia tu negocio.

**Tweet 4:** El destino es la relación directa. La lista de email, la newsletter, el producto que resuelve el problema real. Ahí vive el negocio.

**Tweet 5:** Usa las plataformas para que te encuentren. Construye en tierra propia para que se queden. La distinción entre los dos lo cambia todo.
