# Habilidades como activos, no como credenciales

La mayoría de las personas usa sus habilidades para conseguir empleo. Una minoría las usa para construir activos. La diferencia no está en las habilidades en sí. Está en cómo las tratan.

Una habilidad usada para empleo produce un salario que para cuando paras de trabajar. La misma habilidad convertida en un curso, una consultoría estructurada, un producto descargable o un servicio de retención produce ingresos que se acumulan con el tiempo. La habilidad no cambió. El vehículo cambió. Y el vehículo es lo que determina el techo de tus ingresos y el nivel de control que tienes sobre tu tiempo.

El proceso para convertir una habilidad en un activo tiene tres pasos. Primero, identifica el problema específico que esa habilidad resuelve mejor que la mayoría de las personas. Segundo, encuentra el grupo de personas que tiene ese problema y tiene dinero para resolverlo. Tercero, crea un producto o servicio estructurado que entregue la solución de forma repetible. No necesitas hacer esto con diez habilidades. Con una bien empaquetada es suficiente para empezar a cambiar tu ecuación financiera.
