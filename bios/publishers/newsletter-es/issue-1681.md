# El precio que filtra

La mayoria de los emprendedores independientes tienen un problema de precio. No cobran poco porque no tienen confianza. Cobran poco porque todavia piensan que el precio es una peticion. No lo es. El precio es una declaracion. Le dice al mercado quien eres, para quien trabajas y que tan en serio te tomas tu propio tiempo.

Cuando subes el precio, ocurre algo contraintuitivo. Los clientes malos desaparecen solos. Los buenos clientes, los que quieren resultados reales, preguntan menos y confian mas. El margen que ganas no solo es financiero. Es mental. Tienes espacio para pensar, para entregar bien, para decir no cuando algo no encaja.

El precio correcto no es el que consigue el mayor numero de clientes. Es el que atrae al tipo correcto. Y el tipo correcto, bien servido, se convierte en referencia. Eso vale mas que cualquier descuento. Sube el precio antes de sentirte listo. La confianza llega despues, no antes.
