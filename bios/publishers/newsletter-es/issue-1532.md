# Cómo hacer que el trabajo difícil sea más fácil

El trabajo difícil no se vuelve más fácil porque esperas. Se vuelve más fácil porque lo haces repetidamente hasta que deja de ser difícil.

## El error de buscar motivación primero

La motivación es consecuencia, no causa. Viene después de empezar, no antes.

Cuando esperas sentirte motivado para hacer el trabajo difícil, confías en una variable que no controlas. Cuando creas el sistema que te hace empezar independientemente de cómo te sientes, controlas el resultado.

Los profesionales no esperan sentirse inspirados. Aparecen y hacen el trabajo. La inspiración aparece mientras trabajan, no antes.

## El poder del umbral mínimo

El obstáculo más grande no es el trabajo en sí. Es el inicio.

Una vez que empiezas, la inercia te lleva. El problema es el momento de transición entre no trabajar y trabajar.

El umbral mínimo reduce ese obstáculo al absurdo. "Solo voy a escribir una oración." "Solo voy a abrir el documento." "Solo voy a revisar mis notas por cinco minutos."

Noventa por ciento de las veces, esos cinco minutos se convierten en una hora de trabajo real.

## Diseña el entorno para el trabajo difícil

La fuerza de voluntad es escasa. El diseño de entorno es ilimitado.

Si el trabajo difícil requiere concentración, diseña un entorno donde la concentración sea la opción predeterminada. Teléfono fuera del cuarto. Notificaciones apagadas. Un solo documento abierto.

Si tienes que hacer un esfuerzo activo para distraerte, te distraes mucho menos.

## Acción de esta semana
Identifica el trabajo más difícil de tu semana. Programa un bloque de 45 minutos para hacerlo primero mañana. Prepara el entorno la noche anterior: sin distracciones, con lo que necesites a mano.
