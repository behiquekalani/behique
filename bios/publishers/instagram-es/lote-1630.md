## Post 3
**El output sin input eventualmente se vacía. El input sin output es acumulación sin propósito.**

El ciclo es lo que importa. Leer, procesar, crear, publicar.
No hay un orden correcto universal. Hay el orden que funciona para ti.
Encuentra el ciclo y protégelo.

Ciclo activo, mente activa.
