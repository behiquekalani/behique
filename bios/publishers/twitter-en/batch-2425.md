# Twitter Batch 2425

1. 90 focused minutes beats 8 scattered hours. Your brain needs uninterrupted time to do anything real.

2. Every morning: what is the one thing that makes everything else easier or unnecessary? Do that first.

3. Multitasking is rapid task-switching with a cognitive penalty each time. You're not faster. You're shallower.

4. A blocked calendar is a protected brain. Deep work first. Noise second. Always.

5. The to-do list is mostly noise dressed up as urgency. Real progress comes from one thing done completely.
