# Twitter Batch 2190

## Tweet 1
Every system has one binding constraint.

Fix it and everything moves.

Optimize anything else and nothing changes.

## Tweet 2
Great product, no audience. The constraint is distribution.

More product work changes nothing.

Audience work changes everything.

## Tweet 3
Most people work on what they're comfortable with.

Writers write more.
Builders build more.

This feels productive. It accomplishes nothing if it's not the constraint.

## Tweet 4
The question to ask every week:

"What is slowing me down the most right now?"

Not "what should I do more of?"

Where is the actual bottleneck?

## Tweet 5
Removing one constraint can double output without adding hours.

That's the highest-leverage move available at any stage.

Find it. Fix it. Find the next one.
