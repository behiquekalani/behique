# Twitter Batch 2431

1. Building in public earns trust before you launch. Building in private earns confidence. Pick based on your stage.

2. Some ideas get killed by early feedback. If you need momentum first, build privately.

3. Every post about what you're building is a trust deposit. By launch day, some people are already bought in.

4. Build privately when you need confidence. Go public once the core is real enough to get useful feedback.

5. The path matters less than the destination. Both approaches end with a product in front of people.
