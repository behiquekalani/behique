1. The market does not pay for effort. It pays for outcomes. These are not the same thing and confusing them is expensive.

2. Hard work pointed at the wrong problem is still waste. Intensity without direction is not a virtue.

3. Before asking "am I working hard enough," ask "am I working on the right thing." The second question changes everything.

4. Value is defined by the person receiving it, not the person delivering it. Build with that in mind.

5. If nobody would pay for the outcome, more hours will not fix the fundamental mismatch. Change the problem you are solving.
