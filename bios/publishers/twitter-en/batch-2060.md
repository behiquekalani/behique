1. The first version does not need to be good. It needs to exist. Good is built through iteration, not planning.

2. Perfectionism is procrastination wearing better clothes. The standard is not the problem. The delay is.

3. Version one gets you version two. Version two gets you version three. You cannot skip to the good version. You have to earn it.

4. Ship ugly. Ship incomplete. Ship before you are confident. The feedback you get from shipping is worth more than the comfort of waiting.

5. Done and shipped beats perfect and pending every time. The world only sees what you release.
