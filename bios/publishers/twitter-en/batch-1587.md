# Twitter Batch 1587

## Tweet 1
The best creative tool is the one you use consistently.

Not the one with the most features.

## Tweet 2
Every tool you add has a maintenance cost, a learning curve, and a switching cost.

More tools equals more overhead.

Overhead is the enemy of output.

## Tweet 3
Contrarian take: most creators would produce better work with fewer apps, not more.

One capture tool. One writing tool. One task list.

The empty space is where the thinking happens.

## Tweet 4

How many actually did?

## Tweet 5
Digital minimalism is not about using old tools.

It is about being intentional. Every tool should earn its place by solving a real problem you have right now.

Delete before you add.
