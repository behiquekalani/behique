# Twitter Batch 1455

## Tweet 1
The only productivity metric that matters: did you move the thing that matters most today?

Not "was I busy." Not "did I clear my inbox."

Did the important thing move.

## Tweet 2
Every morning, before anything else, write down the one thing that would make today a success if it were the only thing you finished.

Then protect it.

## Tweet 3
Reactive work will fill every available minute if you let it.

Proactive work requires a decision. Make the decision before the day makes it for you.

## Tweet 4
If you track hours, you optimize for presence.
If you track output, you optimize for results.

Which metric is actually running your day?

## Tweet 5
What would you accomplish this week if you could only ship one thing?

Start with that. The rest is negotiable.
