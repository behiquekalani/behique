# LinkedIn Post 1847

Una sola fuente de ingresos no es un negocio. Es un punto único de falla.

El solopreneur que solo tiene servicios está a un mes malo de entrar en pánico. El que tiene servicio, curso y biblioteca de productos digitales necesita tres problemas simultáneos para tener un mes malo. La acumulación importa.

Empieza con lo que puedes vender esta semana. Agrega la siguiente fuente cuando esa sea estable. No intentes construir todo a la vez. Lleva una cosa a $1,000 al mes y luego construye la siguiente. Para el año dos tienes un portafolio que no colapsa cuando una pieza falla.

Diversifica ingresos como los inversores diversifican portafolios.
