# LinkedIn Post 1815

No necesitas otra herramienta. Necesitas usar la que ya tienes.

El hábito de coleccionar herramientas es procrastinación con disfraz profesional. Cada nueva app que descargas es un retraso para empezar. Cada flujo de trabajo que lees es una excusa para no correr el flujo imperfecto que ya tienes.

El mejor calendario de contenido es el que realmente usas. El mejor sistema de notas es el que tienes abierto ahora mismo. La mejor herramienta de gestión de proyectos es la que no te deja esconderte del trabajo.

Deja de investigar sistemas. Empieza a usar el que tienes, mal, hasta que funcione.
