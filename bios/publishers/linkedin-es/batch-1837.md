# LinkedIn Post 1837

No necesitas una gran idea. Necesitas una solución específica para un dolor específico.

Las grandes ideas son para inversores. Las soluciones son para clientes. La persona que te pagará tiene un problema específico que no la deja dormir. No busca una visión. Busca alivio.

Cuanto más precisamente puedas nombrar su dolor, con las palabras exactas que ellos usan para describirlo, más obvio es que los entiendes. El entendimiento crea confianza. La confianza crea ventas.

Encuentra el dolor. Nómbralo exactamente. Construye el alivio.
