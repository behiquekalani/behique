# LinkedIn Post ES — batch-1598

Los sistemas no tienen que ser complicados para funcionar.

Una lista de verificacion es un sistema. Un bloque recurrente en el calendario es un sistema. Una plantilla que reutilizas cada semana es un sistema.

El objetivo no es la sofisticacion. El objetivo es la consistencia sin fuerza de voluntad.

No deberias tener que motivarte para hacer la misma tarea de la misma manera cada vez. De eso se encarga el sistema.

Empieza con tus tareas mas repetidas. Escribe cada paso. Cronometrate mientras las haces. Luego averigua que pasos podrian ser mas rapidos, eliminados o agrupados.

La mayoria omite esto porque parece tedioso. Pero pasar una hora documentando un proceso que haces diez veces por semana te ahorra tiempo por el resto de tu vida.

Los sistemas tambien te hacen delegable. Cuando un proceso esta escrito, alguien mas puede ejecutarlo. Asi es como eventualmente transfieres trabajo operativo sin perder calidad.

Construye sistemas simples. Ejecutalos consistentemente. Mejorales cuando fallen.

Los sistemas simples le ganan a los complejos cada vez porque los simples realmente se usan.
