# LinkedIn Post 1803

La consistencia no se trata de disciplina. Se trata de eliminar la decisión.

La disciplina es finita. Se agota a las 3 de la tarde. Las personas que parecen consistentes sin esfuerzo no tienen más disciplina que tú. Diseñaron un sistema que elimina la fricción entre la intención y la acción.

No deciden escribir cada mañana. Se sientan, abren el archivo, y empiezan. La decisión se tomó una sola vez, semanas atrás, cuando construyeron el sistema. Ahora simplemente corre solo.

Cada hábito que quieres mantener necesita un disparador, un camino de mínima resistencia, y una salida por defecto. Construye el sistema una vez. Deja que te cargue.
