05 -- YouTube SEO

YouTube SEO

Title Formula

Include primary keyword in first 5 words. Under 60 characters.

Description Template

First 2 lines visible before "Show more". Include keyword + CTA.

SEO Checklist (per video)
  • Primary keyword researched (volume + competition)
  • Title includes keyword in first 5 words
  • Description: 200+ words, keyword in first sentence
  • Tags: 5-10 relevant tags, mix broad and specific
  • Custom thumbnail with text overlay
  • Cards and end screens added
  • Chapters/timestamps in description
  • Pinned comment with CTA
