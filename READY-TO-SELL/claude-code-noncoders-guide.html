CLAUDE CODE FOR NON-CODERS

You type. It builds. No coding required.

LESSON 01

What Is Claude Code?

Imagine you could text a really smart friend who can build things for you. You say what you want in plain English. They do it. That is Claude Code. It lives inside a simple window on your computer. You type what you want. It creates it. Websites, documents, organized files, budgets, apps. You never have to learn how any of it works. You just have to say what you need.

How it works, in 30 seconds

You open a window on your computer. You type a sentence like "create a budget tracker for me." Claude Code reads your request, builds the thing, and shows it to you. If you like it, you say yes. If you want changes, you say what to change. That is the entire process.

What you do NOT need to know

  • You do not need to know how to code
  • You do not need to understand programming languages
  • You do not need a computer science degree
  • You do not need to have used a terminal before
  • You do not need to know what a terminal even is
THINK OF IT THIS WAY

Regular Claude chat is like texting a smart friend. Claude Code is like having that smart friend sit down at your computer and actually do things for you. It can create files, organize folders, build websites, and make tools. All you do is describe what you want.

TRY THIS

Think of three things on your computer that annoy you. Maybe your Downloads folder is a mess. Maybe you need a budget. Maybe you wish you had a simple website. Write them down. By the end of this guide, you will know how to ask Claude Code to do all three.

LESSON 02

Installing It

This takes about 5 minutes. You need to do two things: install a small helper program, then install Claude Code itself. Here is every single click and keystroke.

Part 1: Install the helper program

Claude Code needs a small helper program called Node.js to run. Think of it like needing a power outlet before you can plug in a lamp. You install it once and never think about it again.

  1. Open your web browser (Safari, Chrome, whatever you use)
  2. Go to nodejs.org
  3. You will see a big green button that says "Download." Click it
  4. A file will download. Double-click it to open it
  5. Click "Continue" on every screen until it says "Install." Click "Install"
  6. Enter your computer password if it asks. Click "Install Software"
  7. When it says "The installation was successful," click "Close"

Part 2: Install Claude Code

  1. Open Terminal. On a Mac: press Command + Space, type "Terminal", press Enter. On Windows: press the Windows key, type "Terminal", press Enter
  2. You will see a window with a blinking cursor. This is where you type. Do not be intimidated. You are going to type one single line and never need to come back here for setup again
  3. Type this exactly (or copy and paste it):
TYPE THIS
npm install -g @anthropic-ai/claude-code
  1. Press Enter. Wait about 30 seconds. You will see text scrolling. That is normal. It is installing
  2. When it stops and you see the blinking cursor again, it is done

Part 3: Open Claude Code for the first time

  1. In that same window, type claude and press Enter
  2. It will ask you to log in. Follow the link it gives you. Sign in with your Anthropic account (or create one)
  3. Once you are logged in, you will see a prompt where you can start typing requests. That is it. You are in
IF SOMETHING GOES WRONG

If you see any error messages, close the Terminal window completely. Open a new one. Try the install command again. 99% of the time, this fixes it. If it still does not work, go to claude.ai and ask Claude chat "I tried to install Claude Code and got this error" and paste the error message. It will help you fix it.

WHAT TO EXPECT

After installation, every time you want to use Claude Code, you just open Terminal and type claude. That is the only thing you need to remember.

TRY THIS

Install Claude Code right now using the steps above. Open Terminal, type claude, and log in. Once you see the prompt, type "hello" and press Enter. Claude will respond. You just had your first conversation.

Your First Conversation

You are going to build a real website in the next 2 minutes. No, really. You are going to type one sentence, watch Claude Code work, and then open a website that you made.

What to do

  1. Open Terminal. Type claude and press Enter
  2. Type this request exactly:
COPY AND PASTE THIS
Create a simple shopping list website. It should have a text box where I can type items, a button to add them to the list, and a button next to each item to remove it. Make it look clean and modern with a dark background.
  1. Press Enter. Watch what happens. Claude Code will start writing the code for your website. You will see it working in real time
  2. It will ask you to approve creating a file. Type y (for yes) and press Enter
  3. When it is done, it will tell you it created a file. It might say something like "I created shopping-list.html"
  4. Find that file on your computer and double-click it. It will open in your web browser. That is your website

Understanding approve and reject

Sometimes Claude Code will ask for your permission before it does something, like creating a file or changing a file. This is a safety feature. You have three options:

  • y (yes) - Go ahead, do it
  • n (no) - Do not do that
  • Type a message - If you want to explain what to change instead
WHAT TO EXPECT

The whole thing takes about 30 seconds. Claude Code will create one file. When you open it in your browser, you will see a working shopping list. You can add items. You can remove them. You just built a website by typing one sentence.

WANT TO CHANGE SOMETHING?

After it finishes, just tell it what to change. "Make the background blue instead." "Add a counter that shows how many items are on the list." "Make the text bigger." You talk to it the same way you would talk to a person.

Organizing Your Files

Your Downloads folder is a mess. You have hundreds of files with names like "IMG_4382.jpg" and "Document (3).pdf." Claude Code can fix all of this in seconds.

Clean up your Downloads folder

COPY AND PASTE THIS
Look at my Downloads folder. Organize all the files into subfolders by type. Put images in a folder called Images, PDFs in a folder called Documents, videos in a folder called Videos, and everything else in a folder called Other. Do not delete anything.
WHAT TO EXPECT

Claude Code will look through your Downloads folder, create the subfolders, and move every file into the right place. It will show you what it plans to do before it does it, and ask for your approval. Check the plan, say yes, and your folder will be organized in seconds.

Rename photos by date

COPY AND PASTE THIS
Look at all the photos in my Downloads/Images folder. Rename each one using the date it was taken, in this format: 2024-03-15-photo-1.jpg, 2024-03-15-photo-2.jpg, and so on. If you cannot find the date, use today's date.
WHAT TO EXPECT

Claude Code will read the date information stored inside each photo, then rename them all in order. You will go from "IMG_4382.jpg" to "2024-03-15-photo-1.jpg" which is actually useful and searchable.

Find specific files

COPY AND PASTE THIS
Find every PDF file on my computer. List them all with their full location and file size. Sort the list by size, biggest first.
WHAT TO EXPECT

Claude Code will search your entire computer and give you a list of every PDF, showing you where each one lives and how big it is. This is incredibly useful for finding that one document you downloaded months ago and forgot about.

IMPORTANT

Always include "Do not delete anything" in your request when organizing files. Claude Code will ask for permission before moving files, but adding this to your request makes sure nothing gets lost.

LESSON 08

Tips and Tricks

After using Claude Code for a while, you start to notice what works best. Here is everything I have learned about getting the best results.

How to give better instructions

VAGUE (WORSE RESULTS)

  • "Make me a website"
  • "Organize my stuff"
  • "Help with my data"
  • "Write something for work"

SPECIFIC (BETTER RESULTS)

  • "Make a recipe website with search and categories"
  • "Organize my Downloads folder by file type into subfolders"
  • "Read expenses.csv and tell me my top spending category"
  • "Write a 3-paragraph email declining a project offer politely"
THE FORMULA

Good instructions follow this pattern: What you want + how it should look or work + where to save it. Example: "Create a to-do list (what) with categories, due dates, and priority levels (how) and save it as todo.html on my Desktop (where)."

What to do when it makes a mistake

Claude Code is not perfect. Sometimes it misunderstands what you want. Here is how to handle it:

  1. Be specific about what is wrong. Do not say "that is not right." Say "the chart is showing the wrong months" or "the button should be on the left side, not the right"
  2. Ask it to undo. Say "undo that last change" or "go back to how it was before"
  3. Start fresh if needed. If things get messy, say "let us start over. Here is what I want from scratch" and describe it again more clearly
  4. Say no. When it asks for approval and the plan does not look right, say no and explain what you want instead

When to use Claude Code vs regular Claude chat

USE CLAUDE CODE WHEN

  • You need to create or edit files
  • You want to build something (website, tool, app)
  • You need to organize files on your computer
  • You want to analyze files or spreadsheets
  • You are working on a multi-step project

USE REGULAR CLAUDE CHAT WHEN

  • You just need a quick answer
  • You want to brainstorm ideas
  • You need help writing text (without saving to a file)
  • You want to have a conversation or get advice
  • You are on your phone

Five tricks that make everything better

  1. Say "dark theme" or "modern design." Claude Code defaults to basic-looking results. Adding these words makes everything look 10 times better
  2. Say "save the data" or "remember my entries." If you are building a tool like a tracker, this makes it keep your data between sessions
  3. Ask for changes one at a time. Instead of listing 10 changes, ask for them one by one. You get better results and can approve each one
  4. End with "save it as [name].html on my Desktop." This way you always know where your files end up
  5. Build on what you have. Once Claude Code makes something, you can keep improving it. "Add a search bar." "Change the colors." "Add a print button." Each conversation can build on the last
