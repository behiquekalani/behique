03 -- Risk Management

Risk Management

Position Sizing Rules
  • Never risk more than 5% on one position
  • Set stop-loss before entering
  • Use dollar-cost averaging for entries
  • Keep 10-20% in stablecoins
  • No leverage above 3x
  • Never invest money you need short-term
Red Flags Checklist
  • Anonymous team with no track record
  • Guaranteed returns promised
  • Token unlock cliff approaching
  • Declining on-chain activity
  • Single exchange liquidity
  • No working product, only hype

07 -- Security

Security

Wallet Security Checklist
  • Hardware wallet for long-term holds
  • Seed phrase stored offline (metal backup)
  • Never share seed phrase with anyone
  • Use separate wallets for DeFi vs. holding
  • Enable 2FA on all exchanges
  • Use unique email for crypto accounts
Scam Prevention
  • Verify URLs before connecting wallet
  • Never click links from DMs
  • Revoke unused token approvals
  • Use a burner wallet for minting
  • Bookmark exchange and DeFi sites
  • Check contract addresses on Etherscan
