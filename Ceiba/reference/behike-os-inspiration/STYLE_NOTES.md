# Behike OS Design Inspiration
# Source: Game UI screenshots (cyberpunk/sci-fi)
# Saved: 2026-03-21

## Elements to incorporate (Kalani's picks):

### From ref1.jpg (purple/cyan holographic):
- Purple and cyan color palette
- Holographic/3D feel
- HUD overlay elements (coordinates, status bars)
- "SEARCHING..." and "MOLECULAR DISASSEMBLY" style status text
- Curved screen/visor effect

### From ref2.jpg (blue circuit board):
- Dark blue circuit board grid as background
- "LOADING MEMORY" with progress indicator
- Japanese katakana as decorative elements
- Blue neon wireframe aesthetic
- "ESTEP READY" style boot/status messages

### From ref3.jpg (green terminal + lore panel):
- Green monospace text on dark background
- Scrolling ticker bars at top and bottom
- Split layout: sidebar + main content area
- Version numbers and system identifiers
- Detailed content panels with illustrations
- "RUNNERS' GUIDE" style headers with separator lines

## How to merge with Kalani's Apple style:
- Keep Apple's clean layout and whitespace
- Use pixel font (Press Start 2P) for system chrome, status bars, tickers
- Use system font for body content (readability)
- Add scrolling ticker as a subtle element (not overwhelming)
- Circuit board grid as VERY subtle background (2-3% opacity)
- Neon accents as one of the theme options
- Boot sequence inspired by "LOADING MEMORY" screen
- Status bar with system info (like the game's HUD)
- The virtual desktop IS the unique twist (not just another terminal site)

## DO NOT copy:
- The exact green terminal look (too generic, "that guy" already did it)
- Japanese text (not Kalani's culture, use Taino references instead)
- The game's specific UI layout

## DO incorporate Taino elements instead:
- Taino symbols as decorative elements (like the Japanese katakana)
- Machine names from Taino mythology (Ceiba, Cobo, Hutia)
- "BEHIKE SYSTEM" instead of "CYBERACME"
- Puerto Rico coordinates in the HUD
