## Script 1280: How to Think Like a Systems Designer
**Hook (0-3s):** "Stop solving the same problem twice."
**Body (3-45s):**
1. Every time you do something manually for the second time, you have a systems problem.
2. Systems thinkers ask: how do I solve this once so it runs without me?
3. Content: build templates. Client work: build processes. Finances: build automations.
4. One hour building a system saves 100 hours running the manual version.
5. Start with your most repeated task. Document every step. Then automate or delegate each one.
6. You don't scale with more effort. You scale with better systems.
7. The business you want lives on the other side of the systems you haven't built yet.
**CTA (45-60s):** "Follow for the builder's mindset. What's one system you need to build this week?"
**Caption:** The goal isn't to work harder. It's to build systems that work without you. Think like an architect. #systemsthinking #productivity #solopreneur #businessautomation #buildersmindset
