# Boletin Issue 1364

**Asunto:** Aprende una habilidad en semanas, no en anos

---

Dos personas empiezan a aprender lo mismo el mismo dia.

Una termina el curso en tres meses y aun no ha hecho nada con la habilidad.

La otra deja el curso a los cinco dias, construye algo feo con lo que aprendio, y en un mes ya tiene un proyecto funcional.

La segunda persona aprendio mas. No porque dedicara mas tiempo, sino porque ciero el ciclo.

---

**El insight**

El aprendizaje real no ocurre cuando consumes informacion. Ocurre cuando aplicas informacion y enfrentas friccion real.

El error clasico: acumular conocimiento hasta sentirse listo para actuar. La trampa es que el sentimiento de estar listo casi nunca llega solo con el estudio.

Llega cuando empiezas y te das cuenta de que puedes manejar lo que surge.

La formula que funciona:
1. Aprende el minimo para empezar. No el maximo para perfeccionar.
2. Construye algo con eso. Lo mas rapido posible.
3. Choca con los problemas reales.
4. Vuelve a aprender solo lo que necesitas para resolver lo que chocaste.

Ese ciclo es mas rapido, mas profundo y mas duradero que cualquier curso.

---

**Accion semanal**

Elige una habilidad que llevas semanas o meses "aprendiendo."

Define el proyecto mas pequeno posible que usaria esa habilidad.

Inicia ese proyecto esta semana. No cuando termines el modulo. Esta semana.

---

La habilidad no se adquiere estudiandola. Se adquiere usandola.

Sigue construyendo.
