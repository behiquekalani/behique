# Post 38 - Behind The Scenes: The Tech Setup
## Caption
People always ask what my actual setup looks like. Here it is.

Hardware: MacBook M4 as my main machine. A secondary desktop with a GTX 1080 Ti for running local AI models. Both connected through a custom bridge system.

Software: Claude as my primary AI partner. Cursor for coding. n8n for automations. Warp as my terminal. Obsidian for notes. Git for version control on everything.

The secondary machine runs Ollama for local model inference. Anything that needs GPU processing goes there. Strategy and writing stays on the Mac.

Total investment in the setup: reasonable. The Mac was the big expense. The desktop was built from parts over time. Most software is free or open source.

The point is not the specific tools. It is that every tool has a clear job. No overlap. No redundancy. Every piece earns its place.

Build your stack with intention. If a tool does not save you time or make you money, remove it.
