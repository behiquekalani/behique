# LinkedIn Post 1388 (ES)

Elimine mi lista de tareas el ano pasado.

No metaforicamente. Borre la app y deje de usarla.

Porque una lista de tareas es una coleccion de posibilidades. No te dice que hacer a continuacion.

La reemplace con tres cosas:

1. Una MIT (Tarea Mas Importante) por dia. Una. No tres. No cinco. Una.
2. Una revision semanal cada domingo que decide las MITs para la semana.
3. Una lista "algun dia" que captura todo lo demas sin saturar mi atencion.

El resultado: hago menos cosas y termino mas de ellas.

La productividad no se trata de rendimiento. Se trata de completitud.

Una cosa terminada tiene valor. Una cosa comenzada pero no terminada casi no tiene ninguno.

El cerebro humano esta cableado para sentir las cosas incompletas mas fuertemente que las completadas. Eso se llama efecto Zeigarnik. Tus proyectos a medias estan drenando energia que podrias usar en la unica cosa que importa.

Una cosa. Hecha. Luego la siguiente.

Intenta esto una semana y cuentame que pasa.

Cual es la unica cosa que realmente cambiaria tu trayectoria si la terminaras esta semana?
