# LinkedIn Post 1391 (ES)

La mejor inversion que hice este ano no fue un curso.

Fue construir un sistema que elimino un problema recurrente.

Cada semana pasaba 3 horas haciendo el mismo trabajo manual. Pase un fin de semana automatizandolo. Eso es todo.

3 horas cada semana durante 52 semanas son 156 horas al ano.

156 horas que ahora paso construyendo en lugar de manteniendo.

La mayoria de las personas conoce esta matematica. La mayoria nunca lo hace porque "ya llegare a eso."

Despues no llega. El trabajo manual si.

Elige una tarea recurrente que te cueste mas de una hora a la semana. Automatizala, delegala o eliminala este mes.

No el proximo trimestre. Este mes.

La primera automatizacion se paga sola en pocas semanas. La disciplina de hacerlo una vez construye el habito de hacerlo repetidamente.

Asi es como recuperas tu tiempo sin ganar mas dinero.

Cual es la tarea que haces manualmente cada semana que sabes que deberia estar automatizada?
