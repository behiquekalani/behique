## The Checklist

### 1. Ollama (replaces ChatGPT Plus)
Run LLMs locally on your own hardware. Llama 3.1, Mistral, Gemma. No API costs, no rate limits, full privacy. Works on Mac, Linux, Windows.
- Get it: ollama.com

### 2. n8n Self-Hosted (replaces Zapier)
Visual workflow automation. Connect any API, build complex automations, run on your own server. The self-hosted version is completely free.
- Get it: n8n.io

### 3. Obsidian (replaces Notion for notes)
Local-first markdown notes. No cloud dependency, instant speed, plugin ecosystem. Your data stays on your machine.
- Get it: obsidian.md

### 4. Cursor (replaces GitHub Copilot)
AI-powered code editor with free tier. Built on VS Code, understands your codebase, generates and refactors code.
- Get it: cursor.com

### 5. Claude Code (replaces expensive dev agencies)
CLI agent that reads, writes, and ships code. Handles full-stack tasks from terminal. One of the most underrated tools in the stack.
- Get it: claude.ai/code

### 6. Playwright (replaces paid scraping tools)
Browser automation by Microsoft. Scrape any site, automate form fills, run headless. Python, JS, or .NET.
- Get it: playwright.dev

### 7. Cloudflare Pages (replaces paid hosting)
Free static site hosting with global CDN. Deploy from GitHub. Custom domains, SSL, unlimited bandwidth on the free tier.
- Get it: pages.cloudflare.com

### 8. DaVinci Resolve (replaces Premiere Pro)
Professional video editing, color grading, VFX, and audio. The free version handles everything most creators need.
- Get it: blackmagicdesign.com

### 9. GIMP + Photopea (replaces Photoshop)
GIMP for desktop editing, Photopea for browser-based PSD editing. Both free, both capable. Covers 90% of design tasks.
- Get it: gimp.org / photopea.com

### 10. Telegram Bots (replaces paid notification tools)
Build custom bots for alerts, task logging, idea capture. Python + python-telegram-bot library. Free, instant, runs anywhere.
- Get it: core.telegram.org/bots
