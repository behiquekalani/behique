# Content Repurposing System
# 1 piece of content = 10+ pieces across platforms
# Maximize output, minimize creation time

---

## THE REPURPOSING MATRIX

Every piece of content can be transformed into multiple formats:

### From a YouTube Video (long-form):
1. YouTube video (original)
2. Blog post (transcribe + edit)
3. Instagram carousel (extract 5-7 key points)
4. Instagram Reel (30-60 second clip)
5. TikTok video (same clip, different caption)
6. X/Twitter thread (key insights as tweets)
7. Newsletter section (summary + personal take)
8. Quote graphic (1 powerful sentence)
9. Pinterest pin (key stat or tip)
10. LinkedIn post (professional angle on same topic)

### From a Blog Post / Guide:
1. Blog post (original)
2. Instagram carousel (section by section)
3. Twitter thread (bullet points become tweets)
4. Newsletter feature
5. YouTube script (expand into video)
6. Reel (quick tip version)
7. Quote graphic
8. Infographic (data/process visualization)

### From a Reel / TikTok:
1. Short video (original)
2. Instagram Story (with poll or question)
3. YouTube Short
4. Tweet with video
5. GIF for comments/replies
6. Blog post (expand the concept)
7. Newsletter mention

## WEEKLY REPURPOSING WORKFLOW

### Sunday: Create the Source Content
- Write 1 YouTube script OR 1 in-depth blog post
- This is the "content seed" for the week

### Monday: Extract and Transform
- Pull 5 carousel topics from the source
- Write 2 reel scripts from the source
- Draft 1 Twitter thread from the source
- Write newsletter section from the source

### Tuesday-Saturday: Post Across Platforms
- Each day: 1 IG post + 1 TikTok + 2 tweets + 1 story
- All from the same source content, different formats

### Result:
- 1 source piece -> 7 IG posts + 5 TikToks + 1 thread + 10 tweets + 5 stories + 1 newsletter
- That's 29 pieces of content from 1 creation session

## AI-ASSISTED REPURPOSING PROMPTS

### Extract Carousel Topics:
"Extract the 5 most shareable insights from this content. For each, give me: a hook (1 sentence), the key point (2-3 sentences), and a CTA."

### Convert to Thread:
"Turn this content into a Twitter thread. Hook tweet first (must stand alone and make people want to read more). Then 5-7 tweets, each making one point. End with a CTA tweet."

### Write Reel Script:
"Write a 60-second reel script from this content. Start with a hook that stops scrolling. Deliver one clear insight. End with 'save this' or 'follow for more.'"

### Newsletter Summary:
"Summarize this content in 3-4 sentences for a newsletter. Add a personal take that feels like an insight, not just a recap."

### Quote Graphic:
"Extract the single most powerful sentence from this content that would work as a standalone quote graphic."

## CONTENT LIFECYCLE

```
Week 0: Create source content
Week 1: Post primary versions (IG, TikTok, YT)
Week 2: Post secondary versions (threads, newsletter, blog)
Week 4: Reshare best performers with new hooks
Week 8: Update and re-release evergreen content
Week 12: Compile into a compilation or guide
```

Content that's 3 months old is new to 90% of your audience. Don't be afraid to reshare.
