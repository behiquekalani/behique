## POST 3 . Claude Code Tutorial: Your First CLAUDE.md (Carousel)
**Account:** @behikeai
**Format:** Carousel (6 slides)

---

**Slide 1 (Hook):**
Claude Code does not know who you are.

Until you write a CLAUDE.md. That file changes everything.

**Slide 2 . What CLAUDE.md is:**
CLAUDE.md is a markdown file you put in your project folder.

Claude Code reads it at the start of every session. It is your standing instruction set. Your project context. Your rules.

Without it, you explain yourself every session. With it, Claude already knows.

**Slide 3 . What to put in it, section 1:**
Start with identity.

Who are you. What is this project. What stack are you using. What stage is this at.

Six lines. That is all this section needs. Claude will stop asking basic questions.

**Slide 4 . What to put in it, section 2:**
Add rules.

What Claude should never do in this project. Code style preferences. Naming conventions. File structure. How you want errors handled.

Be specific. Vague rules produce vague behavior.

**Slide 5 . What to put in it, section 3:**
Add context.

Current status of the project. What is working. What is broken. What the next milestone is.

Update this section weekly. A two-minute update saves you ten minutes of re-explaining every session.

**Slide 6 . The result:**
Claude Code with a good CLAUDE.md feels like a developer who has been on the project for months.

No repeated context. No wrong assumptions. No re-explaining your stack.

## POST 4 . I Automate 80% of My Content with n8n (Reel Caption)
**Account:** @kalaniandrez
**Format:** Reel Caption

---

Voice memo to Instagram post in 8 minutes.

This is the exact workflow. No fluff.

Step 1: I record a voice memo on my phone. Raw idea. Unfiltered. 60-90 seconds.

Step 2: The file uploads to a watched folder. n8n detects it and triggers the workflow.

Step 3: OpenAI Whisper transcribes the audio. The transcript goes into a Claude prompt. Claude turns it into a draft post in my voice.

Step 4: The draft lands in a review folder. I read it, edit if needed, approve. Takes 2 minutes.

Step 5: Buffer picks it up and schedules it.

That is the whole loop. Voice memo in, scheduled post out. The only step I own is the 90-second recording and the 2-minute review.

The rest runs without me.
