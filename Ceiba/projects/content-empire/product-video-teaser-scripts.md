# Product Video Teaser Scripts
# Screen-recording demos with voiceover. No face needed.

---

## FORMAT NOTES

Each teaser follows the same structure:
- Opening: grab attention immediately. State the problem or the result.
- Demo: show the actual product, not a slideshow about the product.
- Close: one clear CTA. Where to get it.

Recording tip: use OBS or QuickTime for screen capture. Keep cursor movements slow and deliberate. Dark mode on everything. Font size up. No browser notifications visible.

Voiceover tip: record in one take when possible. Slight imperfections are fine. Reads flat if too rehearsed.

## PRODUCTION CHECKLIST (per video)

- [ ] Screen recorded in dark mode
- [ ] Browser bookmarks bar hidden
- [ ] Notifications disabled
- [ ] Font size 16+ for all readable text
- [ ] Cursor slow and deliberate
- [ ] No background noise in voiceover
- [ ] Voiceover matches screen timing (record together or sync in edit)
- [ ] End card: Gumroad URL + "Link in bio" text overlay
- [ ] Caption for accessibility (auto-captions or manual)
- [ ] Thumbnail: product cover on dark background, 1-line description
