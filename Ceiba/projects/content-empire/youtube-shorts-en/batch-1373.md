# YouTube Shorts EN — batch-1373

**Topic:** Why your to-do list never gets done

**Script:**

Your to-do list never gets done because it is a wish list, not a plan.

Problem one: too many items. A daily list with more than five tasks is a recipe for failure. You will always add more than you can finish. Cut to five. Non-negotiable.

Problem two: no time estimates. "Write blog post" means nothing. "Write 500-word intro section: 30 minutes" is actionable. Estimate every task in minutes.

Problem three: no priority order. You default to easy tasks first. The hardest, most important thing sits at the bottom and never gets done. Put it at the top. Do it first.

A to-do list is a productivity tool, not a storage system. Use it like a tool. Subscribe for more systems that get real results.
