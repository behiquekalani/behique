# YouTube Shorts ES — lote-1411

**Hook:** Contenido gratuito primero. Producto de pago despues. Siempre.

**Punto 1 — Por que importa la secuencia.**
Vender antes de haber construido confianza es como proponer matrimonio en la primera cita. Tecnicamente posible. Casi nunca funciona. La confianza se construye a traves del valor entregado repetidamente sin condiciones.

**Punto 2 — Cuanto gratis es suficiente.**
Cuando las personas empiezan a preguntar donde pueden pagarte por mas, has dado suficiente gratis. Esa pregunta es la señal. No un numero establecido de publicaciones. No un conteo de seguidores. La pregunta.

**Punto 3 — Hacer la oferta.**
Cuando vendes, el producto de pago debe sentirse como el siguiente paso natural, no como un giro. "Te encanto la version gratuita? Aqui esta la version mas profunda." Esa transicion debe sentirse inevitable, no discordante.

**CTA:** Suscribete para el plan completo de negocios de creador construido sobre la confianza primero.
