# YouTube Shorts ES — lote-1263

**Tema:** Productos digitales — imanes de leads que convierten
**Duracion:** ~62 segundos

---

Un iman de leads es el recurso gratuito que regalas para construir tu lista de correo. La mayoria los hace mal. Aqui esta como hacerlo bien.

El error: regalar algo generico. "Guia gratuita de productividad" no atrae a nadie. Hay miles de esas.

Punto uno: tu iman de leads debe resolver un problema muy especifico completamente. No una vision general. Una solucion completa a un problema concreto. "La lista de verificacion exacta de rutina matutina que use para pasar de cero a $3K al mes trabajando medio tiempo" es especifica. Cuenta una historia. Convierte.

Punto dos: debe ser inmediatamente utilizable. Una lista de verificacion, una plantilla, un archivo de referencias. No un ebook de 40 paginas que nunca van a leer. Cuanto mas rapido sea el valor, mayor es la calidad percibida.

Punto tres: debe apuntar hacia tu producto de pago. El recurso gratuito debe resolver parte del problema. Tu producto de pago resuelve el todo. El iman es el punto de entrada, no el destino.

Pon este iman en cada plataforma. Lleva trafico a una pagina simple de registro. Construye la lista.

Esa lista es tu activo de negocio mas valioso.

Suscribete para el recorrido completo de creacion de imanes de leads.
