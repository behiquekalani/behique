# LinkedIn Post 1386

The day I stopped trying to go viral and started trying to be useful was the day my content actually started growing.

Viral content gets attention. Useful content builds trust.

Attention evaporates. Trust compounds.

Here's the test I run on everything I write now:

"Would someone bookmark this and come back to it?"

If yes, I publish it. If no, I rewrite it.

Bookmarks tell you something follows told you: this person found something worth saving.

The content that has built my audience isn't the posts that blew up. It's the posts that kept getting shared six months later because they answered a real question.

Write for the person who needs the answer. Not for the algorithm.

The algorithm notices when real people actually engage with your work over time.

Start with the human. The distribution follows.

What's the most useful thing you could write this week for the person you're actually trying to reach?
