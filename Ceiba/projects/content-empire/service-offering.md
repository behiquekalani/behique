# AI Agent Installation Service

## The Rebrand
Old name: "n8n AI Agent Business"
New name: **"Behique AI" or "AI Agent Installation Service"**

The old framing was wrong. n8n is a tool, not a product. Nobody buys "n8n." They buy "AI employees that work while you sleep."

## What We Actually Built (proof of concept)
- 6 autonomous AI agents running on 3 networked computers
- Auto-run pipeline: task created, auto-assigned, auto-executed, completed without human touch
- Git worktree isolation per task (no agent corrupts another's work)
- Task queue with bridge servers across machines
- CLI spawner using Claude Max (zero API cost)
- Full theme system, decision inbox, yoloMode for auto-approval

This isn't theory. It works. We tested it. Tasks complete end-to-end.

## The Offer (Hormozi $100M Offers framework)

### Dream Outcome
"Your business has AI employees that handle [specific task] 24/7 while you focus on growth."

- Dream Outcome: Automation of repetitive tasks (HIGH)

- Hiring an employee: $40,000-60,000/year
- Hiring a contractor: $5,000-15,000/month
- Our AI agent installation: $500-2,000 one-time + $200/mo support
- ROI is obvious in month 1

## Service Tiers

### Tier 1: AI Audit (FREE - lead magnet)
- 30-minute call or async audit
- Identify 3 tasks AI could automate in their business
- Deliverable: 1-page audit report with ROI estimates
- Purpose: Build trust, qualify leads, feed pipeline

### Tier 2: Starter Agent ($500)
- 1 AI agent installed for 1 specific task
- Examples: email sorting, content scheduling, data entry, customer FAQ
- Includes: setup, testing, 30-day support
- Timeline: 48 hours

### Tier 3: Agent Team ($2,000)
- 3-5 AI agents working together
- Custom workflow design
- Dashboard for monitoring
- Includes: 90-day support, 1 revision
- Timeline: 1-2 weeks

### Tier 4: Full Autonomy ($5,000+)
- Complete autonomous pipeline (what we built for ourselves)
- Multi-machine setup
- Auto-assign, auto-execute, auto-approve
- Includes: 6-month support, training session
- Timeline: 2-4 weeks
