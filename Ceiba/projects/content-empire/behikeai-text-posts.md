## BH-3 [Quick Tip]

AI tip: Run your own LLM for free.

1. Install Ollama (ollama.com)
2. Run: ollama pull qwen2.5:7b
3. Run: ollama run qwen2.5:7b
4. You now have a private AI assistant. No API key. No monthly fee. No data leaving your machine.

Works on Mac, Linux, and Windows.

## BH-9 [Comparison]

Local AI vs Cloud AI:

Local: Free, private, offline, slower

For personal projects and content creation, local wins.
For production apps serving thousands, cloud wins.

The smart move: prototype local, deploy cloud.
