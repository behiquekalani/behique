# Behike Discord Server Plan

## Why Discord (not Circle, not Telegram)
- Free. Circle is $39/mo minimum.
- Gumroad has built-in Discord integration. One toggle.
- Every free starter blueprint download = auto Discord invite.
- Channels can gate by role (free vs paid customers).
- Voice channels for live Q&A later.
- Bot ecosystem for automation.

## Server Name
**Behike Builders**

## Channel Structure

### PUBLIC (everyone sees)
- #welcome - auto-greeting, rules, what this server is
- #introductions - new members introduce themselves
- #announcements - new products, updates, drops
- #free-resources - links to free tools, starter blueprint, Sanctuary app

### BUILDERS (main community)
- #general - open conversation
- #wins - share your wins, no matter how small
- #accountability - daily check-ins, what you're working on
- #product-feedback - get feedback on your products before launch
- #show-your-work - screenshots, links, progress updates

### BY NICHE (one channel per blueprint niche)
- #ecommerce
- #freelancing
- #content-creation
- #ai-automation
- #youtube
- #newsletter
- #dropshipping

### TOOLS
- #tech-stack - what tools people use
- #ai-tools - Claude, ChatGPT, n8n, automation talk
- #design - Canva, Figma, cover design, branding

### SUPPORT
- #product-help - questions about Behike products
- #bug-reports - issues with web apps (Lumina, Sanctuary, etc)

### VOICE
- #coworking-lounge - silent coworking (cameras off, just presence)
- #office-hours - weekly live Q&A (schedule TBD)

## Gumroad Integration
1. Create Discord server
2. Go to server Settings > Widget > Enable
3. Copy the invite link
4. In Gumroad product page, paste the Discord invite URL in the "Invite your customers to a Discord server" field
5. Every purchase/download auto-sends the invite
