**The Vibe:** Raw, punchy, bedroom house. Heavily reliant on drum machines and hardware compression.

## B. Production Techniques

- **The "Pump" (Sidechain Compression):** They routed their entire mix through the Alesis 3630 compressor and used the kick drum to trigger it. This makes the whole track volume dip every time the kick hits.
    
- **Hardware Filtering:** Sweeping the cutoff knob on an analog low-pass filter to slowly bring elements into the mix.

## 1. The TR-909 Kick Drum (Homework style)

This creates that deep, punchy house kick.

JavaScript

```
function play909Kick(audioCtx, destination, time) {
    const osc = audioCtx.createOscillator();
    const gain = audioCtx.createGain();
    
    osc.connect(gain);
    gain.connect(destination);
    
    // Pitch envelope: Starts high, drops fast for the "punch"
    osc.frequency.setValueAtTime(150, time);
    osc.frequency.exponentialRampToValueAtTime(45, time + 0.1);
    
    // Amplitude envelope: Quick attack, medium decay
    gain.gain.setValueAtTime(1, time);
    gain.gain.exponentialRampToValueAtTime(0.01, time + 0.5);
    
    osc.start(time);
    osc.stop(time + 0.5);
}
```

## 2. The Juno-106 Synth Lead (Discovery style)

Combines a Sawtooth and Square wave with a lowpass filter.

JavaScript

```
function createJunoSynth(audioCtx, destination) {
    const sawOsc = audioCtx.createOscillator();
    const squareOsc = audioCtx.createOscillator();
    const filter = audioCtx.createBiquadFilter();
    const vca = audioCtx.createGain();
    
    sawOsc.type = 'sawtooth';
    squareOsc.type = 'square';
    
    // Slight detuning for analog thickness
    squareOsc.detune.value = 10; 
    
    filter.type = 'lowpass';
    filter.frequency.value = 1000; // Cutoff knob
    filter.Q.value = 5; // Resonance knob
    
    sawOsc.connect(filter);
    squareOsc.connect(filter);
    filter.connect(vca);
    vca.connect(destination);
    
    return {
        playNote: (freq, time) => {
            sawOsc.frequency.setValueAtTime(freq, time);
            squareOsc.frequency.setValueAtTime(freq, time);
            vca.gain.setValueAtTime(0.8, time);
        },
        stopNote: (time) => {
            vca.gain.exponentialRampToValueAtTime(0.01, time + 0.2);
        },
        start: () => {
            sawOsc.start();
            squareOsc.start();
        }
    };
}
```

## 3. The Alesis 3630 Sidechain "Pump" Effect

This is how you make the whole track duck when the kick hits.

JavaScript

```
function setupSidechain(audioCtx, trackAudioNode, kickAudioNode, destination) {
    // Create the compressor
    const compressor = audioCtx.createDynamicsCompressor();
    compressor.threshold.setValueAtTime(-30, audioCtx.currentTime);
    compressor.knee.setValueAtTime(10, audioCtx.currentTime);
    compressor.ratio.setValueAtTime(12, audioCtx.currentTime);
    compressor.attack.setValueAtTime(0.003, audioCtx.currentTime);
    compressor.release.setValueAtTime(0.25, audioCtx.currentTime); // The "pump" speed

    // Route the main track through the compressor
    trackAudioNode.connect(compressor);
    compressor.connect(destination);
    
    // In Web Audio, to sidechain we route the kick into the compressor's parameters 
    // (Note: Native sidechain routing requires a slightly more complex work-around with waveshapers in plain Web Audio, 
    // but applying an envelope to a gain node on the track bus mimicking the kick is the standard Web Audio hack).
    
    // Simplified Sidechain Hack for Web App:
    const sidechainGain = audioCtx.createGain();
    trackAudioNode.connect(sidechainGain);
    sidechainGain.connect(destination);
    
    return {
        triggerPump: (time) => {
            sidechainGain.gain.setValueAtTime(0.1, time); // Volume drops
            sidechainGain.gain.exponentialRampToValueAtTime(1.0, time + 0.3); // Volume swells back up
        }
    };
}
```
