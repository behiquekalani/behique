# Ceiba Inbox

Drop files here from ChatGPT or other AI tools. Claude Code reads them and executes.

## How to use

1. Brainstorm with ChatGPT (free, unlimited)
2. Ask ChatGPT: "Format this as a clean markdown file for my developer tool"
3. Save the output here as: chatgpt-[topic].md
4. Tell Claude Code: "Read Ceiba/inbox/chatgpt-[topic].md and build it"

## Naming convention

- chatgpt-bios-phase1.md (build spec)
- chatgpt-product-idea.md (product concept)
- chatgpt-research-polymarket.md (research findings)
- chatgpt-content-strategy.md (strategy doc)

## After Claude Code processes a file

Move it to Ceiba/inbox/processed/ so you know it was handled.
