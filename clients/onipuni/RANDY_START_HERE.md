# RANDY START HERE

Your AI toolkit for Oni-Puni. Follow these steps in order.
Total time: about 30 minutes. You only do this setup once.

---

## STEP 1: INSTALL YOUR AI TOOLS (10 min)

1. Go to https://ollama.com
2. Click the big "Download" button
3. Install it like any other app (drag to Applications on Mac, run the installer on Windows)
4. Open Terminal (Mac) or PowerShell (Windows)
   - Mac: press Command + Space, type "Terminal", hit Enter
   - Windows: press the Windows key, type "PowerShell", hit Enter
5. Type this and hit Enter:

```
ollama pull llama3.1:8b
```

6. Wait for it to finish downloading (takes a few minutes)

Done. You now have a free AI running on your computer. No monthly fee, no subscription.

---

## STEP 2: GENERATE YOUR FIRST PRODUCT DESCRIPTION (5 min)

1. Open Terminal or PowerShell (same as Step 1)
2. Go to your tools folder. Type this and hit Enter:

```
cd onipuni-tools
```

3. Run this command (replace the details with your own product):

```
python3 product_writer.py "Rilakkuma Plush" "San-X" "40.00" "Soft brown bear with overalls, 14 inches"
```

4. The AI will write a full product description for you
5. Copy the output text
6. Paste it into your Shopify product description

That is it. You just wrote a professional product listing in 30 seconds.

## STEP 3: GENERATE A WEEK OF SOCIAL MEDIA (10 min)

1. Open Terminal or PowerShell
2. Go to your tools folder:

```
cd onipuni-tools
```

3. Run this:

```
python3 social_batch.py
```

4. Open the folder called `output/weekly-content/`
5. Inside you will find:
   - 7 Instagram post captions (ready to copy-paste)
   - 7 TikTok video ideas with scripts
   - 3 Pinterest pin descriptions

You now have a full week of content ready to go.

1. Open Terminal or PowerShell


## STEP 5: START TIKTOK (do this today)

This is the highest-impact thing you can do right now. No tools needed, just your phone.

1. Create a TikTok account with the username @onipuni
2. Film yourself opening ONE blind box
   - Prop your phone up vertically
   - Good lighting (near a window works)
   - Show your reaction, be genuine
3. Post it with one of the captions from your AI-generated content (Step 3)
4. Do this every single day for 30 days

Why this matters: blind box openings are one of the most viral formats on TikTok. The kawaii community is massive there. One video can bring more traffic than a month of Instagram posts.
