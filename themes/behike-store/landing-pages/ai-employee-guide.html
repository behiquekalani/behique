The backstory

I was paying $200/month for cloud AI. Then I looked at the old laptop in my closet.

February 2026. I'm a comp-sci student in Puerto Rico. I had three AI subscriptions running, two cloud API bills, and a growing feeling that I was doing this wrong.

$200 a month. For what? Running the same models I could run locally on hardware I already owned.

What this won't do

Honesty saves us both time.

  • It won't match GPT-4 quality. Local models are good. They're not frontier-tier. The guide is honest about where they fall short.
  • It won't work without a GPU. If your only computer is a MacBook Air, you'll need to buy something. The guide covers what to look for under $200.
  • It won't set itself up. You're going to open a terminal, type commands, and debug errors. That's the point. You learn the system by building it.
  • It won't replace your cloud tools overnight. I still use Claude for some things. The goal is reducing dependency, not religious purity.
  • It won't teach you machine learning. This is systems engineering. Connecting things, routing tasks, making hardware useful. Not training models.
