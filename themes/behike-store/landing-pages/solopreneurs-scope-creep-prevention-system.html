How it reads inside

02

The Discovery Call Question

The question that surfaces hidden assumptions: "When this project is complete and you consider it a success, what does it include and what doesn't it include?" The second part is what most solopreneurs skip. It is the most important thing to establish before you write a proposal.

03

Proposal Language

Every proposal should include a section called "Not included in this scope." List three to five things specifically not included. Clients who see "copywriting not included" will tell you immediately if they expected copywriting. That conversation is infinitely easier to have before signing than after.

05

Change Request Workflow

When a client requests something outside the original scope, do not respond to the request directly. Respond to the recognition that a request has been made. "Let me capture this and get back to you with what it would take to include it." This pause communicates that additions go through a process.

08

Project Close Process

Every project should have a formal delivery moment, not a gradual trailing off. Written acceptance is the moment the project officially ends. Without it, the project end date is ambiguous, and ambiguity creates room for post-delivery requests to be framed as project deliverables.

Scripts included

Four conversations, fully scripted

Script 1 — First time a client requests something outside scope

References the document, not your memory. Offers a clear next step. Sets a specific time. Does not say "no." Says "here is the process."

Script 2 — Client pushes back on a change request

Invites both parties to look at the project brief together. The document against the document, not your memory against theirs.

Script 3 — Client escalates

Offers a specific option while maintaining the documentation requirement. You are not refusing. You are redirecting toward a path that keeps both parties clear.

Script 4 — Post-delivery when a client requests additional work

Short. Warm. Clear. The project is closed. New work is new scope. Opens the door to a new proposal without friction.
