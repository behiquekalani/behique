FAQ

Common questions.

Do I need to know how to code to use this? +
No. You need to be able to open a terminal, run a pip install command, and paste an API key into a config file. The setup guide walks through every step with screenshots. If you have ever followed a README before, you will be fine.
What does the script actually require to run? +
Python 3.8 or higher, a few standard libraries (all installed via pip), and a free eBay developer API key. The API key takes about five minutes to set up on the eBay developer portal. No paid API tiers required for basic usage.
How does the script find product opportunities? +
It pulls recently sold eBay listings in your target categories, cross-references active listings to identify pricing gaps, and scores each product on demand signals, competition density, and estimated margin. The scoring framework guide explains exactly how each field is calculated so you can tune the weights to your sourcing strategy.
Does this work for Shopify or only eBay? +
The data source is eBay. The output is a product opportunity list with scoring and margin estimates. That list is equally useful for sourcing decisions on Shopify, Facebook Marketplace, or any other platform. The script itself is eBay-specific because that is where the public sold data lives.
