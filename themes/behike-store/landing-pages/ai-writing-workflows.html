THREAD PROMPT Twitter / X 
You are writing as [your name/persona], a [brief description] who writes about [topics].

Voice brief: [paste your voice brief]

Task: Write a Twitter/X thread about [topic].

Angle: [your specific take, the thing you believe that most people don't say]

Format:
- 8-10 tweets
- First tweet is the hook (bold claim, no questions)
- Each tweet stands alone but builds toward the conclusion
- Final tweet is a direct call to action
- No hashtags. No emojis. Short sentences.

Key points: [your specific points]
Do not include: hype language, generic advice, filler phrases
EDIT LOOP PROMPT Any format 
Remove all hedging language from this paragraph. Be direct.
Do not use phrases like "it's worth noting," "it's important to remember,"
or "some might argue."

---

This section sounds too formal. Rewrite it in a more direct,
conversational tone. Keep all the same information.

---

This is too long. Cut it by 30% without losing the main point.
Remove filler. Keep specifics.
REPURPOSING PROMPT One piece, 10 outputs 
Here is a [blog post / newsletter]. I want to repurpose it for multiple platforms.
The core idea is: [state it in one sentence].

My voice brief: [paste it]

Create the following:
1. Twitter thread (8 tweets, start with the strongest hook)
2. LinkedIn post (300-500 words, professional tone)
3. Instagram caption (hook in first line)
4. TikTok/Reel script (60-75 seconds, one takeaway)
5. Pinterest pin headline + description

Keep the core argument consistent. Rewrite for each platform.
Do not copy-paste sections.

Source content: [paste here]
