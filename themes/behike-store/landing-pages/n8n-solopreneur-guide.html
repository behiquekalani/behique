Workflow Preview

This is what a lead capture workflow looks like in n8n.

Visual logic. No code. Each node is a tool you already use.

n8n — lead-capture-workflow.json
// Workflow: Lead Capture + Auto Follow-Up
// Trigger: New Typeform Submission

[TRIGGER] Typeform → Watch for new responses
↳ Fields: name, email, company, message

[ACTION] Google Sheets → Append row to "Leads" tab
↳ Columns: timestamp, name, email, company, status=new

[CONDITION] IF company field is not empty
↳ TRUE: send B2B follow-up template
↳ FALSE: send B2C follow-up template

[ACTION] Gmail → Send email (personalized, instant)
↳ To: {{ $json.email }}, Subject: "Hey {{ $json.name }}, got your message"

[ACTION] Slack → Send notification to #leads channel
↳ Message: "New lead: {{ $json.name }} from {{ $json.company }}"

// Total execution time: ~1.2 seconds
// Manual steps replaced: 4
// Time saved per lead: ~8 minutes
