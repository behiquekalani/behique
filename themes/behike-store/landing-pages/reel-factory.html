FAQ

Common Questions

Do I need to film anything?
No. The pipeline produces text-based reels and voiceover-style content. You can use your own face later, but the default output is faceless: topic, captions, music, graphics. No camera required.
How does the music work if it is not stock audio?
Background tracks are generated using the Web Audio API. Each one is synthesized in real-time from parameterized patterns (tempo, tone, texture). Every track is mathematically unique. No library. No licensing. No strikes.
What niches does this work for?
Any niche with RSS feeds or active subreddits. Currently optimized for: AI tools, solopreneur/creator content, finance, and tech. Other niches can be configured with the right source setup.
Can I see examples of the output?
Yes. Send an email and request a sample pack. We will generate 3 sample scripts for your niche, with captions and music specs, before you commit to anything.
Is the Content Kit hard to set up?
It requires basic comfort with Python and the terminal. If you can run a script and follow a README, you can set it up. If not, the Pipeline Install tier handles setup for you.
